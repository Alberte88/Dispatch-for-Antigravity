Since you don't use Telegram or Discord much, a custom Web UI might be the most satisfying. Select an option below to compare details:

Premium Web UI (Responsive Mobile Web App)

A custom dark-mode dashboard tailored for mobile browsers. Securely exposed to your phone via a free Cloudflare Tunnel.

Cons: Needs a background tunneling service running on your desktop.
B

Telegram Bot

Control your agent by sending text messages to a private Telegram bot.

Pros: Zero local network configuration needed, native mobile push notifications out-of-the-box.
Cons: You don't use Telegram much; UI is constrained to chat bubbles and basic inline buttons.
C

Discord Bot

Control your agent via commands in a private Discord server channel.

Pros: Native mobile notifications, easy text commands, no local port-forwarding.
Cons: You don't use Discord much; UI is limited to Discord's chat layout and message formatting.

How Security & Tunneling Works for Option A

Cloudflare Tunnels (Free & Secure): A small official agent (cloudflared) runs on your local machine. It connects outbound to Cloudflare, mapping your local port (e.g. localhost:3000) to a public URL (e.g., https://my-dispatch.trycloudflare.com).

Authentication: To prevent random people from triggering tasks on your computer, we can secure the Web App with a simple password, passwordless magic link, or a secure token stored in your mobile browser's cookies.
